02 / 05 WEB VERSION

웹사이트 버전 — Streamlit Cloud

브라우저에서 기업명 입력 → 15개 에이전트 자동 실행 → 단계별 사람 승인 → 보고서 생성

🏗️ 인프라

  • Streamlit Cloud 배포 · GitHub 자동 CD
  • LangGraph StateGraph 오케스트레이션
  • Anthropic API · claude-sonnet-4-6
  • Supabase — 상태 저장 · 파일 스토리지

📦 결과물

  • 최종 산출: PDF + PPTX + DOCX
  • 분석 히스토리 · 다운로드 링크 자동 생성
  • 실시간 토큰 비용 추적 (에이전트별)
  • KO / EN 이중 언어 UI

⚙️ 파이프라인 흐름

Phase 1 리서치 에이전트 6개 병렬 (2개씩 배치)
👤 리뷰 1 에이전트 출력 확인 → 승인/피드백/중단
Phase 2 종합 분석 — Aggregator · Risk · Strategic
👤 리뷰 2 종합 결과 검토 → 승인/피드백/중단
Phase 3 비평 검토 — Review · Critique · DD Questions
👤 리뷰 3 최종 검토 결과 확인 → 승인/피드백/중단
Phase 4 보고서 구조화 + 작성 → PDF/PPTX/DOCX

Critique 점수 ≥35 & 항목별 ≥7 → Pass · <30 → 재실행 (최대 2회)

로컬 버전 — Claude Code CLI

/dd-local 회사명 한 줄로 실행 · Claude Code 구독만으로 API 비용 $0 · 동일 4단계 파이프라인

🖥️ 인프라

  • Claude Code CLI 내 Agent 서브에이전트
  • 에이전트 14개 · 디스크 기반 데이터 흐름
  • 메인 세션 컨텍스트 격리 (윈도우 소모 최소화)
  • WebSearch + Bash(Python) · Tavily · yfinance

📂 결과물

  • 최종 산출: report.md + report.pdf
  • dd-local-outputs/<slug>/ 에이전트별 JSON 저장
  • 크래시 복구 — 완료 JSON 존재 시 자동 스킵
  • --lang Korean · --docs file.pdf 옵션

⚙️ 파이프라인 흐름

Phase 1 리서치 에이전트 6개 전부 병렬
👤 리뷰 1 출력 확인 → 승인/피드백/중단
Phase 2 Aggregator · Risk · Strategic Insight
👤 리뷰 2 종합 결과 검토 → 승인/피드백/중단
Phase 3 Review · Critique · DD Questions
👤 리뷰 3 최종 검토 → 승인/피드백/중단
Phase 4 6개 섹션 라이터 병렬 → 에디터 조립 → PDF

  • 3단계 Human Checkpoint: 인터랙티브 리뷰
  • 디스크 기반 컨텍스트 격리 → 윈도우 소모 최소화

  • 6개 섹션 라이터 병렬 + 에디터 빠른 조립
  • --lang Korean / --docs file.pdf
