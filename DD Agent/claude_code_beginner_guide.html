02 / 05 STEP 1

설치 & 첫 실행 — 10분이면 끝

필요한 것: 컴퓨터, 인터넷, Anthropic 계정(구독). 코딩 경험은 필요 없습니다.

1

VS Code 설치

code.visualstudio.com에서 다운로드 → 설치. 무료입니다.

2

Node.js 설치

nodejs.org에서 LTS 버전 다운로드 → 설치. Claude Code 실행에 필요합니다.

3

Claude Code 확장 설치

VS Code 왼쪽 확장(Extensions) 탭 → "Claude Code" 검색 → 설치 클릭

4

로그인

VS Code 하단의 Claude Code 패널이 열리면 Anthropic 계정으로 로그인. 월 $20 구독 필요 (Max 플랜).

5

작업 폴더 열기

VS Code에서 파일 → 폴더 열기 → 작업할 폴더를 선택. 이 폴더가 Claude의 "작업 공간"이 됩니다.

핵심 개념: Claude Code는 열려있는 폴더 안에서 파일을 읽고, 만들고, 수정합니다. 폴더 = 프로젝트 = Claude의 작업 범위입니다.
🖥️

VS Code 화면 구성 이해하기

왼쪽 파일 탐색기 — 폴더와 파일 목록
가운데 편집기 — 파일 내용 보기/수정
하단/오른쪽 Claude Code 패널 — AI와 대화하는 곳
하단 바 터미널 — 명령어 입력 (Ctrl+`)
💬

첫 대화 — 이렇게 말해보세요

  • "이 폴더에 있는 파일들이 뭔지 설명해줘"
  • "report.md 파일을 만들어줘"
  • "이 엑셀 데이터를 분석해서 요약해줘"
  • "매주 월요일 보고서 템플릿을 만들어줘"
Tip: Claude에게 한국어로 편하게 말하면 됩니다. 코드 용어를 몰라도 됩니다.

03 / 05 STEP 2

CLAUDE.md — AI에게 주는 "업무 지시서"

프로젝트 폴더 최상단에 CLAUDE.md 파일을 만들면, Claude가 매 대화마다 이 파일을 먼저 읽습니다.
= 새 직원에게 주는 "우리 팀 업무 매뉴얼"이라고 생각하세요.

📋

CLAUDE.md에 넣을 내용

항목설명예시
역할 Claude에게 부여할 역할 "너는 투자 리서치 어시스턴트야"
규칙 항상 지켜야 할 행동 원칙 "항상 한국어로 답변해"
맥락 프로젝트/업무 배경 정보 "이 폴더는 주간 보고서용이야"
출력 형식 원하는 결과물 스타일 "표 형태로, 마크다운으로"
📁 내-프로젝트-폴더/
├── CLAUDE.md ← AI 지시서 (필수!)
├── 📁 .claude/
│   ├── 📁 commands/ ← 스킬 파일 보관
│   │   ├── report.md ← /report 명령
│   │   └── summarize.md ← /summarize 명령
│   └── settings.json ← 설정
├── data.csv
└── 보고서.md

CLAUDE.md 실전 예시 — 그대로 복사해서 쓰세요

# 프로젝트 개요 이 폴더는 주간 뉴스 클리핑 리포트를 작성하는 프로젝트입니다. # 역할 너는 미디어 리서치 어시스턴트야. 기사를 분석하고, 주간 보고서를 작성해. # 규칙 - 항상 한국어로 답변해 - 보고서는 마크다운(.md) 파일로 만들어 - 날짜 형식: 2026-03-12 (YYYY-MM-DD) - 기사 출처(URL)를 반드시 포함해 - 파일명: weekly_report_YYYY-MM-DD.md # 보고서 구조 1. 핵심 요약 (3문장) 2. 카테고리별 뉴스 (표 형태) 3. 시사점 & 액션 아이템 # 참고 - 주요 매체: 한경, 매경, 블룸버그, 로이터 - 관심 분야: AI, 핀테크, 헬스케어
핵심: CLAUDE.md는 그냥 텍스트 파일입니다. VS Code에서 새 파일 → 파일명을 "CLAUDE.md"로 저장하면 끝. 코딩 문법이 아니라 평소 말하듯이 쓰면 됩니다.

04 / 05 STEP 3

Skill (.md) — 반복 업무를 "한 단어"로 실행

자주 하는 작업을 .claude/commands/ 폴더에 .md 파일로 저장하면, /파일명만 입력해서 실행할 수 있습니다.

⚡

Skill이 뭔가요?

  • CLAUDE.md가 "항상 적용되는 규칙"이라면
  • Skill은 "버튼 하나로 실행하는 매크로"입니다
  • 예: /report → "이번 주 뉴스를 분석해서 주간 보고서 만들어"
  • 예: /summarize → "첨부 문서를 3줄로 요약해"
  • 매번 긴 프롬프트를 반복 입력할 필요 없음!
🛠️

만드는 법 (3단계)

1

폴더 만들기

프로젝트 폴더 안에 .claude/commands/ 폴더 생성

2

.md 파일 만들기

그 폴더 안에 원하는 이름으로 .md 파일 생성
(예: report.md)

3

프롬프트 작성

Claude에게 시킬 내용을 평소 말하듯 작성

📄 .claude/commands/report.md — 주간 보고서 자동 생성

오늘 날짜 기준으로 이번 주의 주간 뉴스 클리핑 보고서를 만들어줘. 1. 이 폴더에 있는 기사 파일들을 모두 읽어줘 2. 카테고리별(AI, 핀테크, 헬스케어)로 분류해 3. 각 기사의 핵심을 2줄로 요약해 4. 마지막에 "이번 주 핵심 시사점" 3가지를 뽑아줘 5. 파일명: weekly_report_{{날짜}}.md 로 저장해 보고서 형식은 CLAUDE.md에 정의된 구조를 따라줘.

📄 .claude/commands/summarize.md — 문서 요약

사용자가 지정한 파일을 읽고 요약해줘. 다음 규칙으로 요약해: - 전체 요약: 3문장 이내 - 핵심 포인트: 불릿 5개 이내 - 액션 아이템: 할 일 목록으로 정리 - 원문 길이 대비 요약 비율 표시 $ARGUMENTS 파일을 요약 대상으로 사용해.
사용법: Claude Code 패널에서 /report 또는 /summarize 파일명.pdf 라고 입력하면 끝!
$ARGUMENTS는 슬래시 명령 뒤에 붙이는 텍스트가 자동으로 들어가는 변수입니다.

05 / 05 DAILY OPERATION

매일 쓰는 운영 팁 & 자주 묻는 질문

설치와 설정이 끝났으면, 이제 매일 업무에 활용하는 방법과 알아두면 좋은 팁들입니다.

🎯

핵심 운영 팁 7가지

  • 폴더 = 프로젝트 — 업무별로 폴더를 나누세요. 폴더마다 다른 CLAUDE.md를 쓸 수 있습니다
  • 구체적으로 지시 — "보고서 써줘" 보다 "A4 1장, 표 포함, 한국어, 마크다운"이 훨씬 좋은 결과
  • /help — 모르겠으면 일단 /help 입력
  • 파일 권한 확인 — Claude가 파일을 만들거나 수정할 때 "허용" 팝업이 뜹니다. 내용 확인 후 승인
  • Ctrl + L — 대화 초기화. 새로운 주제로 넘어갈 때 사용
  • Esc — Claude가 작업 중일 때 중단하기
  • 대화가 길어지면 새로 시작 — 20회 이상 주고받으면 성능이 떨어질 수 있습니다
🔑

비용 & 구독

  • Claude Max $20/월 — 기본. 대부분의 업무에 충분
  • Claude Max $100/월 — 헤비 유저. 더 많은 사용량
  • API 방식도 있지만 초보자는 구독(Max)이 간편
  • 구독은 claude.ai/settings에서 관리
❓

자주 묻는 질문

Q. 코딩 몰라도 되나요? 네. Claude에게 "~해줘"라고 말하면 됩니다. 코드는 Claude가 씁니다.
Q. 파일이 날아갈 수 있나요? Claude가 파일을 수정/삭제할 때 항상 확인 팝업이 뜹니다. 꼭 읽고 승인하세요.
Q. 인터넷 검색도 되나요? 네. "~에 대해 웹에서 찾아줘"라고 하면 검색합니다.
Q. 엑셀/PDF 읽나요? 네. 폴더에 넣고 "이 파일 읽어줘"라고 하면 됩니다.
Q. 영어만 되나요? 아니요. 한국어 완벽 지원. 한국어로 말하면 한국어로 답합니다.
